My role included implementing Firebase for enhanced data management, creating a structured reviews.json file for seamless customer review storage, and applying my design skills for an inviting user experience. Global Sky Mosaic

This weather app was a collaborative effort among three developers as part of Holberton's Trimester 2 Hack Sprint. For this project, the tech stack was JavaScript (37.9%), HTML (37.8%), and CSS (24.3%). The app provides up-to-date weather information for various Holberton campuses globally, sourcing data from the WeatherAPI. I was responsible for integrating the API for real-time weather data retrieval, contributing to the project's design, and ensuring a smooth user experience.
